## 🚀 Run Locally
1.Clone the number_bot_website repository:
```sh
git clone https://github.com/Johnny1305/number_bot_website
```
2.Install the dependencies with one of the package managers listed below:
```bash
pnpm install
bun install
npm install
yarn install
```
3.Start the development mode:
```bash
pnpm dev
bun dev
npm run dev
yarn dev
```
